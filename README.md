# PSL - Advanced Nmap + CVE Scanner

> Professional Network & Service Vulnerability Scanner  
> Built with Python + Nmap + NVD API  

---

## 🚀 Overview

PSL (Penetration Scanning Lab) is an advanced network scanning tool that integrates:

- Full port scanning
- Service & version detection
- CVE lookup from National Vulnerability Database (NVD)
- CVSS scoring
- Risk classification
- JSON reporting
- Fast & Full scan modes

This tool is built for **educational purposes and authorized security testing only**.

---

## ⚙️ Features

✔ Fast Scan Mode  
✔ Full 65535 Port Scan  
✔ Service Version Detection  
✔ Automatic CVE Lookup  
✔ CVSS Score Extraction  
✔ Risk Level Classification (Low / Medium / High / Critical)  
✔ Clean CLI Interface  
✔ JSON Report Export  
✔ Professional Output Table
